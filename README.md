# Car Booking System

A Java-based Car Booking application that allows customers to browse, select, and book different types of cars.  
The system supports both console interaction and a JavaFX graphical user interface.

## Features
- Add and manage customers
- View available cars (Ordinary Cars, Taxis, Trucks)
- Book cars and record reservations
- Calculate costs using car characteristics and taxable interfaces
- Read and write booking data using text files
- JavaFX GUI with interactive input fields and actions
- Cleaning service cost options

## Technologies Used
- Java (OOP)
- JavaFX (GUI)
- File Handling (ReadText / WriteText)

## Project Structure
- `Car`, `Taxi`, `OrdinaryCar`, `Trucks` — car types
- `Customer` — customer information
- `Booking` — booking details
- `CarCharacteristics`, `Taxable` — service and cost interfaces
- `CARGUI` — graphical user interface
- `ReadText`, `WriteText` — file I/O support
